# 事件

事件是指用户在某事务上由于某种行为所执行的操作，我们定义了 鼠标事件、渲染事件、组件生命周期事件、模型事件 四种事件类型，分别发生在 鼠标对Component进行操作时，场景渲染中， 场景渲染完成时，组件生命周期改变时 和 模型重新渲染时。 定义了 DTEventopen in new window事件对象，该事件对象记录了触发事件的 Componentopen in new window、js事件对象Event、以及是否阻止事件进一步传播的状态。

# DTEvent 对象

# 实例化

let event = new DTEvent(
  type: ModelMouseEvent | RenderEvent | ComponentLifeCycle | ModelEvent | string,
  options: {
    component?: Component,
    object?: Object3D,
    event?: MouseEvent,
    params?: Record<string, any>
  }
)

  • 参数:

    • type: ModelMouseEventopen in new window | RenderEventopen in new window | ComponentLifeCycleopen in new window | ModelEventopen in new window

    • options: 配置项

      • component: Componentopen in new window

      • object: 3D对象

      • event: js 事件对象 Event

      • params: 额外参数

  • 返回值:

    • DTEvent 实例

# 属性

# stopped

是否阻止事件进一步传播的状态

  • 类型: Boolean

# type

事件类型

  • 类型: ModelMouseEvent | RenderEvent | ModelEvent | ComponentLifeCycle | string

# event

js 事件对象 Event

  • 类型: Event

# component

触发事件的 Component

  • 类型: Componentopen in new window

# params

额外参数

  • 类型: Record<string, any>

# 方法

# stopPropagation()

阻止事件进一步传播

  • 参数:

    • 无

  • 返回值:

    • 无

  • 示例:

event.stopPropagation();

# 鼠标事件

# ModelMouseEvent

鼠标事件发生模型渲染成功后，在组件上对鼠标进行 单击、双击、划过、失焦、移动 操作时。通过监听鼠标事件，可以获取到当前操作的 DTEventopen in new window 对象或 Componentopen in new window 对象。对应的事件名为：

enum ModelMouseEvent {
  dblclick = "dblclick",
  click = "click",
  hover = "hover",
  blur = "blur",
  mousemove = "mousemove",
  contextmenu = "contextmenu",
}

  • dbclick: 鼠标双击

  • click: 鼠标单击

  • hover: 鼠标划过

  • blur: 鼠标失去焦点

  • mousemove: 鼠标移动

  • contextmenu: 鼠标右键

  • 示例:

meta.addListener("dblclick", (event: DTEvent) => {
  console.log("双击的组件", event.target);
  console.log("js事件对象Event", event.mouseEvent);
  event.stopPropagation(); // 阻止事件进一步传播
});

# 渲染事件

# RenderEvent

渲染事件发生在 场景渲染中 和 场景渲染完成时 。通过监听渲染事件，可以获取到当前场景的渲染进度，包括文件总大小、已加载的文件大小、加载百分比。对应的事件名为：

enum RenderEvent {
  loading = "loading",
  loaded = "loaded",
  beforeLeave = "beforeLeave",
  beforeEnter = "beforeEnter"
}

  • loading: 加载中

  • loaded: 加载完成

  • beforeLeave: 离开一个场景前

  • beforeEnter: 进入一个场景前

  • 示例:

meta.addListener(RenderEvent.loaded, ({ path, total, current, percent }) => {
  console.log("文件总大小", total);
  console.log("已加载的文件大小", current);
  console.log("加载百分比", percent);
});

# 组件生命周期事件

# ComponentLifeCycle

组件生命周期事件发生在组件生命周期状态改变时，通过监听该事件，可以准确掌握一个组件的当前状态。

enum ComponentLifeCycle {
  created = "created",
  mounted = "mounted",
  render = "render",
  unmounted = "unmounted"
}

  • created: 创建之后

  • mounted: 挂载之前

  • render: 渲染

  • unmoented:卸载之后

# 模型事件

# ModelEvent

渲染事件发生在执行如 隐藏模型、给模型加勾边 等一系列操作后，会重新渲染模型，通过监听渲染事件，可以获取到执行这些操作的模型。

enum ModelEvent {
  selected = "selected",
  unselected = "unselected",
  multipleSelected = "multipleSelected",
}

  • selected: 模型选中

  • multipleSelected: 模型多选

  • 示例：

meta.addListener("multipleSelected", (obj) => {
  console.log("选中的模型对象数组", obj);
});

meta.addListener("selected", (event) => {
  console.log("选中的component", event.component);
});
