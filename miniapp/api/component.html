# Component

创建 Component 实例

  • 示例
let instance = app.getComponent("radar1");

# 属性

# el

组件 dom

  • 类型：HTMLElement

# uid

组件唯一标识

  • 类型：string

# instance

获取自定义组件实例化后的自定义对象

  • 用于二次开发时直接获取自定义组件暴露的对象

# type

类型：用于区分组件、图层、组、画布等

  • 类型：ComponentType
  • 参考： ComponentTypeopen in new window

# label

组件的中文标签名字

  • 类型：string

# locked

锁定状态

  • 锁定后的组件不可选中

  • 类型：boolean

# version

版本

  • 类型：string

# visible

显示&隐藏

  • 类型：boolean

# parent

父组件

  • 类型：Component

# index

组件下标

  • 类型：number

# top

最外层组件

  • 类型：Component

# action

动作

  • 参考：ComponentActionopen in new window

# script

脚本

  • 参考：ComponentScriptopen in new window

# layer

图层

  • 参考：Layeropen in new window

# animation

动画

  • 参考：Animationopen in new window

# style

样式

  • 类型：Partial<CSSStyleDeclaration>

# attribute

属性

  • 类型：Record<string, any>

# source

数据源

  • 类型：Array<Source>
  • 参考：Sourceopen in new window

# children

子组件

  • 类型：Component[]

# 方法

# search()

根据类型：查询组件列表

  • 参数

    • type: string 类型，包括 component、group、layer、 page

  • 返回值

    • Component[] 查询到的组件列表

  • 示例

let list = instance.search("component"); // 获取所有组件
let list = instance.search("layer"); // 获取所有图层
let list = instance.search("group"); // 获取所有组
let list = instance.search("page"); // 获取画布

# forceUpdate()

强制更新

  • 示例
instance.forceUpdate();

# remove()

移除当前组件

  • 示例
instance.remove();

# removeChild()

移除子组件

  • 参数

    • component: Component | string 组件 / 组件 uid

  • 示例

instance.removeChild("uid");

# insertBefore()

在指定的位置插入组件

  • 参数

    • component: Component | ComponentOptions | Array<Component | ComponentOptions> 组件

    • index?: number 插入位置

    • 参考：ComponentOptionsopen in new window

  • 示例

instance.insertBefore(ComponentOptions, 2);

# appendChild()

在末尾添加组件

  • 参数

    • component: Component | ComponentOptions | Array<Component | ComponentOptions> 组件

    • 参考：ComponentOptionsopen in new window

  • 示例

instance.appendChild(ComponentOptions);

# clear()

清空所有组件

  • 示例
instance.clear();

# createExtension()

创建组件自定义属性实例

  • 参数

    • props?: Record<string, any> 组件

  • 返回值

    • Promise<ReturnType<typeof createExtension> | undefined>

  • 示例

instance
  .createExtension({
    options: instance.$options,
    type: "attribute",
  })
  .then((attribute) => {
    attribute.amount("attribute");
  });

# addEventListener()

监听 dom 事件

  • 参数

    • type: DomEvent 事件类型

    • fn: Function

    • options?: boolean | Record<string, any> 参数

  • 示例

instance.addEventListener("click", (e) => {
  console.log(e);
});

# removeEventListener()

删除 dom 事件

  • 参数

    • type: DomEvent 事件类型

    • fn: Function

    • options?: boolean | Record<string, any> 参数

  • 示例

instance.removeEventListener("click", (e) => {
  console.log(e);
});

# dispatchEvent()

触发一个事件

  • 参数

    • event: Event

  • 示例

instance.dispatchEvent("click");

# createApi()

创建 api

  • 参数

    • api: 定义好的 api 接口

  • 返回值

    • <T extends (...args: any) => any>

  • 示例

import { defineApi } from "@tslfe/miniapp-engine";
const api = defineApi（{
    async getData(id, panelId, params){
        let res = await this.get("/load")
        return res
    }
}
let apiInstance = instance.createApi(api);
apiInstance.getData()

# exportSource()

导出数据

  • 参数

    • key: string 组件包名称

    • name: string | undefined 组件名称

    • data: any 导出数据

  • 示例

let data = instance.exportSource("currentSpace", "空间数据", component.source);

# useSource()

使用数据

  • 参数

    • key: string

      • key格式: componentName:componentKey:sourceKey

      • 例如: "canvas:canvas:currentSpace"

  • 返回值

    • Source[]

  • 示例

let data = instance.useSource("canvas:canvas:currentSpace");

# subscribeSource()

注册数据监听

  • 参数

    • key: string

      • key格式: componentName:componentKey:sourceKey

      • 例如: "canvas:canvas:currentSpace"

    • fn: Function

  • 示例

this.subscribeSource("canvas:canvas:currentSpace", (space) => {
  setup.amount.getData({
    spaceId: space?.data?.id,
  });
});

# exports()

导出组件配置

  • 返回值

    • ComponentOptions

    • 参考：ComponentOptionsopen in new window

  • 示例

let Componen = instance.exports();
